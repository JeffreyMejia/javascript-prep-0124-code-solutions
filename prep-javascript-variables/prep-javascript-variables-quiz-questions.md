# prep-javascript-variables-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What does the `=` (equals) operator do in JavaScript?
The `=` operator assigns a value to a variable.
- What operator will return the data type of a variable value?
The `typeof` operator will find the data type of a Javascript variable.
- Which keyword is used to declare a variable in JavaScript?
You declare a variable using the `let` and `var` key words. The `var` is used for older browsers.
- Which characters is a JavaScript variable allowed to begin with?
Identifiers for variables can begin with letters, digits, underscores, and dollarsigns. Underscores and dollar signs are treated as letters in Javascript.
- What are quotation marks used for in JavaScript?
The purpose of quotation marks in Javascript is to write text within strings
- What is the purpose of strings in JavaScript?
The purpose of strings in Javascript is for storing text.
- What is the purpose of booleans in JavaScript?
The purpose of booleans in Javascript is to have a value of either True or False.
- What is the purpose of numbers in JavaScript?
The purpose of numbers un Javascript is to represent values
- What does `null` mean in JavaScript?
`Null` is "nothing"
